# Código de Conducta

Este proyecto adopta el [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) como código de conducta.

## Compromiso

En interés de fomentar un ambiente abierto y acogedor, como contribuyentes y mantenedores nos comprometemos a hacer que la participación en nuestro proyecto y comunidad sea una experiencia libre de acoso para todas las personas.

## Reportar problemas

Reportes de violación → contactar al maintainer vía email privado o GitHub Security Advisories.
